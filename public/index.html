Schedule API

Base URL: /v1, Version: 1.0.0

An API that allows users to interact with shift data

Schemes: http, https

Summary

Path Operation Description
/employee GET

Returns list of all workers within the database
/employee/{emp_id} GET

Return user data by ID
/shift GET

Returns a list of all shifts in the database
/shift/{shift_id} GET

Return detailed shift data by shiftID
/{user_id}/employee POST

Creates user within database
/{user_id}/employee/{emp_id} DELETE

Deletes worker data by ID
PUT

Updates worker data by ID with provided body data
/{user_id}/hours GET

Returns the amount of hours worked within the last 7 days
/{user_id}/schedule GET

Returns a list of shifts that are within start and end time restrictions
/{user_id}/shift GET

Returns a list of user shift and open shift data back to the user
POST

Creates entry for a shift within the database
